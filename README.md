## Project

Implementation of a web interface to a phonebook blockchain application. 

## Getting started

Installing: 

> git clone https://github.com/heronsousa/goledger-challenge-web.git

Go to directory:

> cd goledger-challenge-web

Install dependencies:

> yarn

Start:

> yarn start
